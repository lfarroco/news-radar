The latest release of build2, version 0.16.0, brings exciting new features and improvements for developers. One of the main focuses of this release is the integration with system package managers. Developers can now consume and produce binary distribution packages, making it easier to manage dependencies and distribute their projects. This functionality includes uploading packages directly from CI builds. The release also introduces advanced functionality for more complex projects, such as built-in support for Objective-C/C++ and Assembler with C Preprocessor compilation. Additionally, there are several smaller features and maintenance work, including performance optimizations and improved diagnostics. Check out the release notes for more details and start exploring the new features in build2 0.16.0.

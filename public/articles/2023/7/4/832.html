Inspecting Elixir Dependencies at Runtime for Security

A common challenge in vulnerability response is determining the exact version of a library running in production. Elixir's mix.exs file, like many dependency specification files, does not provide this information. This article by Michael Lubas highlights the difficulties faced by security teams when trying to identify vulnerable versions of libraries. It also introduces a solution called App Audit, a feature in Paraxial.io that automatically records an audit for Elixir applications at runtime. By installing Paraxial.io and configuring the agent, developers can easily track and monitor their application's dependencies. The audit results are uploaded and can be viewed in the "History" tab. App Audit is included with Paraxial.io Application Secure, a comprehensive security and compliance platform for Elixir.

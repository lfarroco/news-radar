MedusaJS releases new Product Module for Node.js developers

June 20, 2023 | Topics: Node.js

MedusaJS, an open source Node.js commerce SDK, has released a new Product Module that brings commerce logic closer to frontend frameworks. The module allows developers to access Products, Variants, Categories, and more through a standalone package that can be installed and run in Next.js functions and other Node.js compatible runtimes, including frameworks like Nuxt, SvelteKit, and Remix, as well as traditional Node applications.

This new deployment mode has many benefits, including the ability to deploy the module independently of the Medusa backend and directly to serverless environments like Vercel. The Product Module is customizable and easily portable into new, modern environments, making it a powerful tool for developers building unique commerce applications.

In addition to the new Product Module, MedusaJS has made general improvements to its commerce SDK. The Recap, a summary of what the company shipped over the past 10 weeks, includes more details about these updates.

For developers looking to build commerce applications, MedusaJS's open source Node.js commerce SDK provides an accessible and customizable solution. The new Product Module is a significant addition to the SDK, enabling developers to access important commerce logic through a standalone package that can be easily integrated into their applications.
